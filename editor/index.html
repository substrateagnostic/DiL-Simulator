How the editor works

    

      Every tab here writes changes to JSON override files in src/data/
      (balance.json, room-overrides.json, character-overrides.json,
      ally-overrides.json, encounter-overrides.json). The game loads its base
      data from JS modules and then merges your overrides on top — so anything you tweak here is a small,
      reviewable diff, never a giant rewrite. Hit Save (or Ctrl+S) on each tab, then refresh
      the dev-server tab in your browser to see it in-game.
    

    

      When you're happy with a batch of changes, click Publish to GitHub. That stages the
      five override files, commits them with a stock message, and pushes to main. The yellow
      dot in the top bar means there are uncommitted edits waiting to be published; the green dot means
      everything's clean. Use the Diff tab to preview exactly what will be committed.
